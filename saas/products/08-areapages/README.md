# AreaPages (Rank #8)

Suburb × service page generator for local SEO.

```bash
npm install
npm run dev      # http://localhost:3007
npm test
npm run build
```

## Features

- 4 demo NSW suburbs with landmarks and council data
- Locally-grounded page generation from job references
- Uniqueness scoring vs existing pages
- Approve → mock WordPress publish workflow
- 5 unit tests
