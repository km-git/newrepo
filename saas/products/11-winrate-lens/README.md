# WinRate Lens (Rank #11)

Win rate analytics for tradies quoting through ServiceM8, Tradify, or Xero.

```bash
npm install
npm run dev      # http://localhost:3010
npm test
npm run build
```

## Features

- 10 demo ServiceM8 quotes with mixed outcomes
- Win rate by job type, suburb, price band, and response time
- Deterministic metrics engine (no LLM for numbers)
- Plain-English monthly insight narrative
- Mock API sync endpoint
- 12 unit tests
