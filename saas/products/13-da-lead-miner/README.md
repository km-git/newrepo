# DA Lead Miner (Rank #13)

Council development-application lead monitoring for builders and trades.

```bash
npm install
npm run dev      # http://localhost:3012
npm test
npm run build
```

## Features

- 3 demo NSW councils (Parramatta, Blacktown, Hills)
- 8 sample DA approvals with work-type classification
- Lead scoring tuned to trade focus (pool builder demo profile)
- Weekly digest with top leads and summary narrative
- Save / review / dismiss workflow
- Mock council feed sync
- 14 unit tests
