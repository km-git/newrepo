# ReviewReply Desk (Rank #9)

On-brand review reply drafting for Google and Facebook with sentiment classification and human approval.

```bash
npm install
npm run dev      # http://localhost:3008
npm test
npm run build
```

## Features

- 4 demo reviews across Google and Facebook
- Sentiment classification (positive / neutral / negative)
- Tone profiles (friendly, professional, warm)
- Draft → edit → approve → mock post workflow
- Escalation flag for negative reviews
- 8 unit tests
