# SubbieCheck (Rank #6)

Subcontractor compliance portal for builders.

```bash
npm install
npm run dev      # http://localhost:3005
npm test
npm run build
```

## Features

- Builder compliance board (3 demo subbies)
- Subbie upload portal (`/portal/sub-001`)
- Certificate text parsing (expiry, policy, insurer)
- Human confirmation before counting toward compliance
- Expiry and missing-document alerts
- 9 unit tests

**Disclaimer:** Tracks documents and dates — not compliance advice.
