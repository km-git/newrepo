# SWMS Studio (Rank #2)

Job-specific SWMS draft generator for tradies and small builders.

```bash
npm install
npm run dev
npm test
npm run build
```

## Features

- Per-trade hazard/control library (electrical, plumbing, carpentry)
- Guided Q&A form → SWMS draft with sign-off blocks
- Keyword-based hazard selection from job description
- Text export (DOCX/PDF planned)
