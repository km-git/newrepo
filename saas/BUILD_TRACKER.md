## Shared stack (all products)

- **Framework:** Next.js 15 App Router + TypeScript + Tailwind
- **Data:** Supabase (prod) / in-memory JSON store (local dev without credentials)
- **Billing:** Stripe (env-gated)
- **LLM:** Provider-agnostic workhorse abstraction in `saas/shared/`
- **Tests:** Vitest per product

## MVP definition

Each product ships with: landing page, core workflow API, dashboard, mock mode, `.env.example`, README, and passing unit tests for business logic.
