# SnapReport (Rank #7)

Site photos + voice notes → branded completion reports.

```bash
npm install
npm run dev      # http://localhost:3006
npm test
npm run build
```

## Features

- Per-trade templates: electrical, pest, solar, maintenance
- Photo captioning (mock) with EXIF strip flag
- Voice note → section structuring
- Report preview and text download
- 6 unit tests
