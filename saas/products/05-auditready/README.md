# AuditReady (Rank #5)

NDIS audit-readiness workspace for small registered providers.

```bash
npm install
npm run dev      # http://localhost:3004
npm test
npm run build
```

## Features

- 9 NDIS Practice Standards indicators (subset)
- Evidence library with auto-matching to indicators
- Staff credential expiry tracking (dates/refs only)
- Gap report generation with readiness score
- 8 unit tests
